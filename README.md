# 📚 MADR - Meu Acervo Digital de Romances  

Desenvolvido com **FastAPI**, o MADR (Meu Acervo Digital de Romances) é uma API voltada para o gerenciamento de livros digitais. Ele permite realizar operações de cadastro, busca, atualização e remoção de romances, oferecendo uma estrutura clara e eficiente para armazenamento e acesso às informações.  

Este código foi desenvolvido como **trabalho de conclusão de curso (TCC)** no curso [FastAPI do Zero](https://fastapidozero.dunossauro.com/), ministrado por Eduardo Mendes, aplicando conceitos fundamentais na construção de APIs modernas e eficientes.      
<br>


## **Ferramentas utilizadas**
- Python 3.12+ 
- FastAPI
- Uvicorn
- Pytest
- SQLAlchemy
- Alembic
- Poetry
- Docker
